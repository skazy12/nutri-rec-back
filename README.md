\# NutriSmart Connect - Backend (Spring Boot)



Backend del sistema de recomendación nutricional. Expone endpoints para:

\- Autenticación (JWT / refresh tokens)

\- Perfil nutricional del usuario

\- Recetas (listado y detalle)

\- Plan nutricional y registro de consumo



\## Requisitos

\- Java 17+

\- Maven

\- PostgreSQL
